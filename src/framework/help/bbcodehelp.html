Links

[url=YOURLINK](YOURTITLE)[/url]

Title and end tag are optional. If YOURTITLE is empty, YOURLINK will be used.

Images

[img=SRC (h=HEIGHT) (w=WIDTH) (float={left|right}) (caption="TEXT")](ALTTEXT)[/img]

Anything in parentheses is optional. Caption styling should be done in the page's CSS, the class is "bbcode_caption".

Objects (Flash/Quicktime)

[object=SRC (h=HEIGHT) (w=WIDTH) (parameters=values)](ALTTEXT)[/object]

Anything in parentheses is optional. You may include extra parameters such as autostart="1" and they will be passed along to the object.

Code Blocks

[code]YOURCODE[/code]

Quotations

[quote(=PERSONQUOTED)]QUOTEDTEXT[/quote]

Headers / Dividers

[h{1-6}]HEADERTEXT[/h{1-6}]
[hr] (closing tag optional)
[clear] (closing tag optional)

Makes sure anything following the [clear] appears below any floating elements (like images).
